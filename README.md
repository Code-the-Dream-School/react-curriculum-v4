# react-curriculum-v4

React curriculum for FY 2026+ (Work in Progress)

## About

This repository contains the React curriculum materials for Code the Dream's Intro to React course. The curriculum is structured as an 11-week course covering React fundamentals through advanced topics like hooks, state management, routing, and deployment.

Key directories:

-   **`/learns-app-content`** - Lesson materials formatted for CTD Learns, Code the Dream's custom Learning Management System (LMS)
-   **`/mentor-content`** - Resources and guides specifically designed for group instructors and reviewers

## Deliverables and Submission Guidelines

To ensure a clear understanding of expectations, please review the definitions for course deliverables and the associated submission guidelines.

### Assignments

Assignments are the core graded components of this course. They are mandatory, crucial for demonstrating mastery of weekly topics, and essential for progression. Each assignment will have specific requirements, grading rubrics, and strict deadlines. Successful completion of all assignments is required to complete the course.

### Exercises

Exercises are supplementary activities designed for practice, skill development, and to reinforce concepts introduced in lessons. They provide hands-on experience and help solidify your understanding. While typically ungraded, diligently working through exercises is highly recommended as they often prepare you for the challenges in assignments. Some assignments may integrate exercises directly into their requirements.

### Important Submission Notes

**Important:** All weekly assignments are subject to instructor review and approval. Ensure submissions meet the stated criteria by their respective deadlines. Late submissions may be accepted with prior arrangement, but timely completion is crucial for staying on track with the curriculum.

## Course Structure & Weekly Breakdown

### Week 1: Introduction to React & Component Basics

**Weekly Deliverables:** For Week 1, complete and submit `Assignment 1: Component Basics`. Practice with `Exercise 1.1: JSX Practice` and `Exercise 1.2: Props and State Basics`.

This week introduces the fundamental concepts of React, including JSX, components, props, and basic state management. You'll learn how to set up a React project and build your first interactive components.

*   **Topics:** Setting up a React environment, JSX syntax, functional components, props, basic state with `useState`.
*   **Learning Objectives:**
    *   Understand the role of React in modern web development.
    *   Be able to create and render functional components.
    *   Pass data between components using props.
    *   Manage simple component state.

### Week 2: State Management with Hooks & Event Handling

**Weekly Deliverables:** This week's deliverable is `Assignment 2: State Management with Hooks`, which includes integrated practical exercises.

Building on the foundation of Week 1, this week delves deeper into React hooks, focusing on `useState` and `useEffect` for more complex state management and side effects. You'll also learn how to handle user interactions and events in React applications.

*   **Topics:** `useState` for complex state, `useEffect` for side effects, event handling, conditional rendering, list rendering.
*   **Learning Objectives:**
    *   Manage complex application state using hooks.
    *   Perform side effects such as data fetching.
    *   Respond to user interactions with event handlers.
    *   Render dynamic lists and conditionally display content.
